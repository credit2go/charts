This site contains helm charts for credit2go kubernetes deploy usage.

It contains two site:

1. candidates repo: https://credit2go.github.io/candidates/ or https://chars.credit2go.cn/candidates/

this repo is in development usage for dev usage.

2. releases repo: https://credit2go.github.io/releases/ or https://chars.credit2go.cn/releases/

this repo is in production ready for each release of application for test and production usage.

Note:

Since we use maven-helm plugin to create charts, it will create charts for each application releases.

The chart version will be same as application version.
